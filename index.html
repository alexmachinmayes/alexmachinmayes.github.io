Refugee Populations In Africa

In 2017, an estimated 2.4 million refugees originated from South Sudan

This interactive map was created help to visualize the origins and relative magnitude of the most extreme refugee populations in Africa. The data contains estimated counts of refugees by country of origin where the number of refugees exceeded 100,000 (1990-2017). The map and graph were developed using the D3 JavaScript library, TopoJSONs, JavaScript, HTM, and CSS.

Declining US Cities

The 40 US cities with the highest percentage of population loss (2010-2017)

Proportional and dynamic city circle markers are utilized to show the range of percent population losses for all cities displayed within the map extent you choose. The map legend shows the maximum and minimum values and the mean of those values as a dynamic reference. The slider control can be used to move through the year ranges (2011 through 2017) illustrating the cumulative population decreases in these cities since 2010. This application was developed using U.S. Census data with Leaflet JS, HTML, CSS, and JavaScript.

Advanced Spatial Analysis

Comparing Nitrate Levels and Cancer Rates in Wisconsin

In this study, advanced spatial analysis methods are used to compare residuals of nitrate levels in well water with cancer rates per census tract in the state of Wisconsin. Inverse distance weighting is used to interpolate a nitrate surface raster from well water points, and ordinary least squares is used to run linear regression analysis. The geoprocessing was conducted using Python, ArcPy and Model Builder. The map visualizations were created using ArcGIS Pro.
